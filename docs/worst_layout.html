Carpalx optimizes keyboard layouts to create ones that require less effort and significantly reduced carpal strain!

Download keyboard layouts, or run the code yourself to explore new layouts.

X11 layouts are available! Patches to include Carpalx layouts in xkeyboard-config and kbd have been submitted by Perry Thompson. Meanwhile, many thanks to Sven Hallberg for providing X11 configuration for Carpalx layouts. Richard Gomes contributed an archive of these files for KDE/Gnome users.

Love your coworkers? Switch them to QWKRFY layout or a fully optimized QGMLWY layout.

Hate your coworkers? Switch them to TNWMLC layout. It's the only keyboard layout that has its own fashion line.

This is typing hell.

Most Difficult Layout

Using knowledge for evil is sometimes a lot more fun than using it for good. By having a model of typing effort, we can ask the question "What layout has the largest effort?".

I applied the effort minimization simulation to QWERTY, but this time asked carpalx to maximize typing effort. In other words, find the layout which is hardest to use. The result is the finger-breaking TNWMLC, which increases effort by almost 90% over QWERTY.

TNWMLC spawns Brazilian Fashion Line

In a collaboration that brings keyboard layout design and fashion together — what a combination! — Brazilian designer Julia Valle used the carpalx worst layout (TNWMLC) as a design element in her Winter 2011 line.

The work was shown in a show in Rio de Janeiro. Julia accented the clothes with beautiful necklaces made out of parts of an 1956 Smith Corona typewriter.

This project is nearly an cosmic alignment of eclectic interests. Keyboard layout research is subverted to generate the most difficult English layout, which is then transformed into beautiful clothing design. This cross-pollination of ideas has particular meaning to me, since I once dabbled in fashion photography.

TNWMLC - detailed statistics

Stroke statistics for TNWMLC are frightening, even when compared to the already suboptimal QWERTY. Hand asymetry is 51% - the left hand is absolutely hammered ... just look at those left hand stroke runs! For ~70% of key strokes, the left hand is used for two or more successive key strokes. The bottom row is used 45% of all strokes and the home row is used only 13% of the time. The pinky is used 34% of the time and the middle finger is extremely underused (9%).
